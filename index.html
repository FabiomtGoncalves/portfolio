ESTIGmail

React, Gmail API

IPB Recicla

Android Studio, JAVA, Machine Learning, Python, Firebase

Aplicação Android para auxiliar e educar a comunidade do IPBeja sobre a reciclagem de diversos objetos como garrafas de plástico, vidro, cartão e metal, e simultaneamente informar os utilizadores sobre o impacto desse objeto reciclado. O principal objetivo desta aplicação é quando um membro do IPBeja reciclar um objeto ao fotografar o mesmo, a aplicação irá reconhecer automaticamente o objeto que está a ser reciclado através de uma rede neural implementada em Python que com a ajuda de Machine Learning (também funciona como um leitor de código de barras com a ajuda do ML Kit do Google) irá reconhecer o objeto.
